# Section Scaffolds

Use these as manuscript structures. They may be filled with AI-assisted working
prose only when disclosure status and direct-submission status remain visible.

## Introduction Scaffold

| paragraph | purpose | evidence needed | author decision |
|---|---|---|---|
| 1 | Define the empirical problem | Institutional fact or motivating descriptive statistic | How broad should the opening scope be? |
| 2 | State unresolved gap | Checked source-evidence cluster and missing comparison | Which literature debate is primary? |
| 3 | Pose research question | Research design note | Exact estimand and population |
| 4 | Describe design and data | Data source, sample, treatment, outcome | Which design facts are safe to foreground? |
| 5 | Summarize result boundary | Preferred table/figure and diagnostics | How strongly to state the causal claim |
| 6 | Contribution map | Checked source-evidence declarations and method contrast | Which contribution matters most to the venue? |

## Literature Review Scaffold

Group by debate, not by paper list:

1. Cluster claim.
2. Papers supporting the cluster.
3. What the cluster identifies well.
4. What remains unresolved.
5. How the current project enters.

For each cluster, include `.aiss` source-evidence ids and citation status.

## Empirical Strategy Scaffold

| block | must contain |
|---|---|
| estimand | Target comparison and population |
| treatment | Variable, timing, treatment group, control group |
| outcome | Variable, unit, transformation |
| model | Equation or design logic |
| inference | Fixed effects, clustering, weights |
| identifying assumption | Parallel trends, exclusion restriction, continuity, ignorability, etc. |
| diagnostics | Pre-trend, balance, placebo, sensitivity |
| threats | What remains unresolved |

## Results Scaffold

For each table or figure:

- What is estimated?
- Which column/panel is preferred?
- What is the unit and scale?
- What uncertainty measure is reported?
- What design boundary limits interpretation?
- Which robustness check speaks to the main concern?

## Mechanism Scaffold

Separate:

- Mechanism directly tested.
- Evidence consistent with a mechanism.
- Mechanism only proposed for future work.

Do not merge these categories.
