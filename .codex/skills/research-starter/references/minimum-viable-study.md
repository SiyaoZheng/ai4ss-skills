# Minimum Viable Study

A minimum viable study is the smallest research object that can teach the researcher whether a route is worth continuing.

It is not a complete paper, not a robustness package, and not a polished argument.

## Patterns

| study type | minimum viable study | first output |
|---|---|---|
| descriptive | One clearly defined population, one measure, one time window | Table shell or figure shell with data gap notes |
| causal | One treatment/exposure, one outcome, one plausible comparison, one threat list | Identification sketch plus required data columns |
| prediction | One target, one feature family, one baseline model, one validation split | Baseline plan and leakage checklist |
| text analysis | One corpus, one coding target, one sampling rule, one human-review slice | Extraction schema and 20-row review sample |
| case comparison | Two to four cases, one comparison axis, one source bundle | Case grid and source sufficiency notes |
| qualitative | One interview/document set, one coding frame, one consent/confidentiality boundary | Coding memo and unresolved ethics questions |
| mixed methods | One quantitative pattern plus one qualitative source path | Bridge memo showing what each side can and cannot answer |
| theory mapping | One concept, one mechanism family, one observable implication | Concept-mechanism-observation map |

## Choosing The Smallest Study

Prefer the route that can answer these questions fastest:

- What is the unit of analysis?
- What material already exists?
- What is the first observable pattern or source fact?
- What would falsify this route's feasibility?
- What decision must the researcher make before deeper work?

## Upgrade Triggers

Move to downstream skills only after the minimum viable study exposes a concrete need:

- Need to build or merge data: `research-data-builder`.
- Need to discover and verify sources: `literature-matrix`.
- Need to audit identification or result claims: `methods-reviewer` or `$did-expert`.
- Need to organize paper sections without drafting prose: `academic-writing-scaffold`.
- Need to present verified artifacts: `research-slides-builder`.
