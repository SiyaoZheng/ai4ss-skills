# Reporting Standards

Use this reference when reviewing tables, figures, slides, or manuscript result claims.

## Regression Tables

Must expose:

- Outcome variable and unit.
- Treatment/exposure variable.
- Sample and period.
- Fixed effects.
- Controls or control groups.
- Standard-error clustering.
- N and number of units/clusters when relevant.
- Model notes explaining transformations and weights.

## Event-Study Figures

Must expose:

- Event time and treatment timing.
- Omitted baseline period.
- Confidence intervals or bands.
- Sample and model family.
- Whether composition changes across horizons.
- Pre-treatment coefficients and uncertainty.

## Descriptive Figures

Must expose:

- Numerator and denominator.
- Units.
- Data source.
- Sample restrictions.
- Whether values are raw, weighted, smoothed, or model-adjusted.

## Result Claims

Check:

- Does the claim name the outcome and population?
- Does it state magnitude or direction with uncertainty?
- Is the claim tied to a specific model/table/figure?
- Does it avoid implying mechanisms that were not tested?
- Does it avoid policy language unsupported by the design?

## Presentation Outputs

Slides should show the evidence object clearly. If the slide only says "significant effect" without model, sample, uncertainty, or source, mark it as insufficient.
